== Description ==

This plugin allows you to add/delete password for all posts or some sepcial posts.

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'WP Posts Password Batch Manager'
3. Activate WP Posts Password Batch Manager from your Plugins page. 
4. Visit 'My PSW ' and set it,that is all.


= From WordPress.org =

1. Download 'WP Posts Password Batch Manager'.
2. Upload the 'wp-posts-password-batch-manager' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate 'WP Posts Password Batch Manager' from your Plugins page. 
4. Visit 'My PSW ' and set it,that is all.
